# Checkbox

基于 Ant Design Checkbox 组件开发，保持与原生组件完全兼容。

## 使用方式

直接使用 Ant Design Checkbox 组件的原有配置项，包括：

- `size`: 控制复选框尺寸
- `disabled`: 禁用状态
- 以及其他所有 Ant Design Checkbox 支持的属性

## 组件特性

- 完全兼容 Ant Design Checkbox 的所有功能
- 支持 CheckboxGroup 等高级使用方式
- 继承 Ant Design 的所有配置项和行为
